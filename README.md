# 🏖️ TravelDream — Туристическое агентство

Демо-сайт многостраничного туристического агентства, созданный на Next.js 16.

## Страницы

- **Главная** — герой-баннер, поиск, статистика, популярные направления, горячие туры, отзывы
- **Направления** — 12 направлений с фильтрацией по континентам и поиском
- **Туры** — 16 туров с фильтрами по категории/сложности, сортировкой, детальным просмотром
- **О нас** — история компании, миссия, команда, таймлайн
- **Контакты** — форма обратной связи, FAQ, контактная информация

## Технологии

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS 4** + **shadcn/ui**
- **Zustand** (навигация между страницами)
- **Framer Motion** (анимации)
- **Prisma** + SQLite (база данных)

## Запуск локально

```bash
# Клонировать
git clone https://github.com/glebgubiev10-del/traveldream.git
cd traveldream

# Создать .env файл
cp .env.example .env

# Установить зависимости
npm install

# Генерация Prisma клиента
npx prisma generate

# Создать базу данных
npx prisma db push

# Запуск dev-сервера
npm run dev
```

Открыть http://localhost:3000

## Деплой на Vercel

1. Импортируйте проект на [vercel.com](https://vercel.com/new)
2. Framework: Next.js (автоопределение)
3. Добавьте переменную окружения: `DATABASE_URL` = `file:./dev.db`
4. Деплой!

## ⚠️ Важно

- **НЕ запускайте** `npm audit fix --force` — это даунгрейдит Next.js с v16 до v9!
- Все версии зависимостей зафиксированы в package.json для совместимости
